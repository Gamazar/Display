Hooks:
    -useState(*default state*): handle reactive data. Handle when data changes
        -state is data
        -when the data changes re-render the UI
        -param: is the default state

        const [count, setCount] = useState(0);
                ^            ^
                |            |
        reactive val        setter

    -useEffect(function) : handles component lifecycle
        -takes in the param a customized function
        -Tells react to run the "effect" function after flushing changes to the DOM
